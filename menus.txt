Menu 1

P - Point Based
C - Category (Percentage) Based
G - GPA Calculator
D - Display Current Save
S - Save to save.txt (NO AUTO-SAVE)
L - Load a Save (NO AUTO-SAVE)
R - HARD RESET (wipe save.txt and/or program memory)
X - Exit Program

	Menu 2-1 (P)

		N - Create a New Class
		A - Add Scores to a Class
		D - Delete Score(s) from a Class
		R - Remove a Class
		L - Display List of Classes (of this type)
		B - Back
		
			ability to stop and cancel (if you haven't started) at any time for all menus
